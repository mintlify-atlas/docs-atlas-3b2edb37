Pi Day is an interactive web application that celebrates March 14th (3/14) — the annual day dedicated to the mathematical constant π. Built with **React 19**, **TypeScript**, **Vite**, **Tailwind CSS v4**, **Framer Motion**, and **shadcn/Radix UI**, the app turns a single scrollable page into an educational and playful experience. Each section introduces a different facet of π — from its geometric definition to its appearances in NASA navigation, signal processing, and quantum mechanics.

## Key technologies

The app is a fully client-side single-page application (SPA). Navigation between sections uses smooth-scroll anchors rather than separate routes — the entire experience lives on one URL. A `wouter`-based router handles the root path and a 404 fallback, while Vercel's rewrite rules ensure direct links always serve `index.html`.

| Layer | Choice |
|---|---|
| UI framework | React 19 |
| Language | TypeScript 5 |
| Build tool | Vite 6 |
| Styling | Tailwind CSS v4 (via `@tailwindcss/vite`) |
| Animation | Framer Motion 12 |
| Component primitives | shadcn/Radix UI |
| Routing | wouter 3 |
| Data fetching | TanStack Query 5 |

## Design philosophy

The visual design follows a dark sci-fi aesthetic: black backgrounds, glowing primary-color accents, subtle white borders at low opacity, and smooth entrance animations driven by Framer Motion. The navbar is transparent at the top of the page and gains a frosted-glass backdrop as you scroll. On mobile, a slide-down menu replaces the horizontal nav links.

<Note>
  Because the app is a single-page application with anchor-based navigation, the URL does not change as you move between sections. Bookmarking a specific section is not supported by design.
</Note>
